Installing RTXI

Installing RTXI can be done in two ways:

Automatic
Use our Debian-based live CDs. It's easier than manually installing RTXI, but differences between your system and the one used to make the live CD could affect real-time performance.

Manual
Install a generic Linux image, compile a Xenomai-patched kernel, and then install RTXI from source. It's more difficult, but it matches RTXI against your own system, unlike the live CD. Try it if you're feeling adventurous!
