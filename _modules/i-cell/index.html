I-Cell

Requirements: None
Limitations: None

Module for a current-clamped cell.

Input Channels

  1. input(0) - Iapp (A)

Output Channels

  1. output(0) - V

Parameters

  1. Iapp_offset - uA/cm^2
  2. rate - Hz

States

  1. m - Sodium Activation
  2. h - Sodium Inactivation
  3. n - Potassium Activation
